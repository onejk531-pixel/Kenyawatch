# Kenya Watch Project

The Kenya Watch project is a new initiative aimed at enhancing the understanding of social issues in Kenya through data-driven insights and visualizations. Our goal is to provide an analytical framework for various datasets, allowing stakeholders to make informed decisions.

## Features
- **Data Analysis**: We perform in-depth analysis of social issues in Kenya, utilizing various datasets.
- **Visualizations**: Our platform offers interactive visualizations that simplify complex data.
- **Community Engagement**: We encourage community feedback and participation in our project.

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/onejk531-pixel/Kenyawatch.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```
